### 3. 가장 막혔던 문제와 해결 과정
- 문제 상황 또는 증상: 오후 4시쯤 피로가 쌓였지만 계속 작업하고 있어서 내가 힘들다는 사실을 제대로 알아차리지 못했다.
- 처음 추측한 원인: 첫날이라 새로운 사람들과 기획을 맞추는 과정에 에너지를 많이 사용했고, 해야 할 일을 한꺼번에 생각하면서 조급해진 것 같다.
- 원인을 확인하기 위해 시도한 방법: 현문님이 밖에 나가자고 제안해 함께 소나무 공원을 산책했다. 작업 자리에서 벗어난 뒤에야 피로한 상태였다는 것을 인식할 수 있었다.
- 실제 원인: 쉬지 않고 계속 집중하려다 보니 현재 컨디션을 확인하지 못했고, 미완료된 GitHub 정리를 머릿속으로만 계속 붙잡고 있었다.
- 해결 방법: 산책으로 잠시 작업과 거리를 두면서 정신을 환기했다. GitHub 정리는 다른 일을 하며 계속 걱정하기보다 따로 시간을 잡아 팀원들과 함께 정리하기로 생각했다.
- 다음에 비슷한 문제가 생기면 먼저 확인할 것: 집중력이 떨어지기 전에 현재 몸 상태를 확인하고 짧게라도 자리를 벗어나겠다. 미완료 작업은 계속 걱정만 하지 않고 담당자와 정리 시간을 먼저 정하겠다.

### 6. 전체 회고
오늘은 실시간 중고 거래 조별 프로젝트를 처음 시작해 기획하고 와이어프레임을 만들려고 노력했다. 첫날이라 아직 팀의 방식과 내가 해야 할 일을 완전히 파악하지 못했고, GitHub 정리가 남아 있다는 생각 때문에 조금 조급했다.
